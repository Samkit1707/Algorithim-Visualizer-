Top Network Security Cheatsheet

1. Physical Layer

The physical layer is the foundation, responsible for the physical transmission of data bits across a network medium. Common attacks include:

  • Eavesdropping/Tapping
  • Physical tampering
  • Electromagnetic interference

2. Data Link Layer

The data link layer packages data into frames for transmission on the same network segment. Common attacks include:

  • ARP spoofing
  • MAC address spoofing
  • Switch flooding

3. Network Layer

The network layer routes packets across networks, determining the best path for data to travel. Common attacks include:

  • IP spoofing
  • Route table manipulation
  • Smurf attack

4. Transport Layer

The transport layer provides reliable data transfer services between applications. Common attacks include:

  • UDP flood
  • SYN flood
  • TCP Hijacking

5. Session Layer

The session layer establishes, manages, and terminates sessions between applications. Common attacks include:

  • Session fixation attacks
  • Session replay attacks

6. Presentation Layer

The presentation layer formats and encrypts data for presentation to the application layer. Common attacks include:

  • SSL Stripping
  • Character Encoding Attacks

7. Application Layer

The application layer provides network services to applications. Common attacks include:

  • SQL Injection
  • Cross-Site Scripting (XSS)
  • DDoS attacks
